# TP2-FSE

Para realização do trabalho é necessario installar na placa as seguinte bibliotecas
smbus2
bme280

Com os comandos 

```bash
pip3 install smbus2
pip3 install RPi.bme280
pip3 install pyserial
pip3 install RPi.GPIO
```
após é entrar na pasta `src`

e rodar com 

Python3 main.py
